# TCC - Pizzaria

Esse é o front-end de um projeto de tcc, na qual estamos desenvolvendo para uma pizzaria.

## Tecnologias utilizadas

- [Stripe](https://stripe.com/br)
- [Vite](https://vite.dev/)
- [React](https://react.dev/)
- [CSS Modules](https://github.com/css-modules/css-modules)

## Como rodar o projeto

```bash
# Clone o repositório
$ git clone https://github.com/MLKP1/Front.git

# Navegue até o diretório do projeto:
$ cd Front

# Instale as dependências:
$ npm install

# Run the app
$ npm run dev
```

## Algumas observações

- Você precisará da extensão Biome para rodar o lint
